## Start from `main`

Every executable Rust program starts in a `main` function. The function body is wrapped in
braces, and each statement inside the body runs from top to bottom.

<Callout type="tip" title="Small pieces first">
  Keep examples compact while learning a new Rust concept. It is easier to reason about
  ownership, values, and output when the program has one visible job.
</Callout>

## Example source

```rust
fn main() {
    let topic = "ownership";
    println!("Next lesson: {topic}");
}
```

## Reading the snippet

- `let` binds a local value.
- `topic` is the local name.
- `println!` writes formatted text to standard output.
